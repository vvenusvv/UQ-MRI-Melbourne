# UQ MRI Melbourne

Monash University 2023 S2
<br>
FIT5032 Internet Applications Development

The homepage for Ultimate Quality MRI Melbourne (UQ MRI Melbourne). Registered customers can make, edit, or cancel an appointment on this web application. They can also retrieve their MRI result and leave a
review through this application. All users can search locations and directions using the service provider map function. In addition, admin users can manage MRI service providers, users and send newsletter to selected registered users.
